From a text file an English text containing letters spaces and dots only words should be extracted. A file dictionary containing the list of words should be created. Each word in the dictionary should be unique repetitions should be eliminated and should be lower case. An algorithm should randomly select words from the dictionary and should compose a simple poem with a used-defined number of words on each line and a used-defined number of lines of a strophe. A synthesized poem should be saved into a text file.
